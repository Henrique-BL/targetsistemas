3º Questão: 
a) 1, 3, 5, 7, ___

Cada número é igual ao anterior mais 2. Portanto, o próximo número é 7 + 2 = 9.

b) 2, 4, 8, 16, 32, 64, ____

Essa sequência é uma progressão geométrica, onde cada termo é obtido multiplicando o termo anterior por 2. 
O próximo número é 64 × 2 = 128.

c) 0, 1, 4, 9, 16, 25, 36, ____

Essa sequência é formada pelos quadrados dos números naturais. O próximo número é 7² = 49.

d) 4, 16, 36, 64, ____

Essa sequência é formada pelos quadrados dos números pares. O próximo número é 10² = 100.

e) 1, 1, 2, 3, 5, 8, ____

Essa sequência é a famosa Sequência de Fibonacci, onde cada termo é a soma dos dois anteriores. O próximo número é 8 + 5 = 13.

f) 2, 10, 12, 16, 17, 18, 19, ____ 

Essa sequência é formada por todos os números que iniciam com a letra d, portanto em ordem crescente o próximo é = 200;
